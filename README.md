# Sistema de Inventario McDonald's - Gestión Ágil (Scrum)

Proyecto de Ingeniería de Software enfocado en la modernización y automatización del control de bodega para una sucursal de McDonald's (San Pedro de la Paz). Este repositorio documenta el ciclo completo de análisis, levantamiento de requerimientos y diseño arquitectónico utilizando el marco de trabajo ágil **Scrum**.

## Objetivo del Proyecto
Migrar el sistema de gestión manual de inventario hacia una solución digital que permita:
* Registrar entradas, salidas, mermas y devoluciones en tiempo real.
* Evitar quiebres de stock mediante alertas automáticas y algoritmos de predicción.
* Agilizar la recepción de productos mediante lectura de códigos de barras (SKU).
* Centralizar la información con control de roles y generación de reportes exportables.

## Metodología y Herramientas
* **Marco de Trabajo:** Scrum (Sprints semanales, Daily Meetings, tableros Kanban).
* **Gestión de Proyecto:** Microsoft Planner.
* **Modelado de Sistemas:** UML (Diagramas de Casos de Uso para Seguridad, Operaciones de Bodega, Control de Stock e Inteligencia/Reportes).
* **Prototipado (UI/UX):** Creación de Mockups para vistas de Administrador, Gestión de Usuarios y Recepción de Stock.

## Mi Rol en el Equipo de Desarrollo
Dentro del Development Team, mis responsabilidades y tareas asignadas (reflejadas en la planificación de los Sprints) incluyeron:
* Diseño del modelo de datos y categorización de productos.
* Análisis y diseño de la lógica para el CRUD de Productos.
* Estructuración del módulo de Inteligencia y Reportes (exportación a PDF/Excel).
* Diseño del algoritmo de predicción de pedidos basado en el historial de ventas.
* Participación en la validación y pruebas piloto.
